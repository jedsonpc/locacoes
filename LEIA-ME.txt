LOCACOES - GESTAO DE EXCELENCIA

Para abrir com melhor funcionamento:
1. Clique duas vezes em abrir-app-locacao.bat ou abrir-app-locação.bat
2. O sistema encontrara uma porta livre e abrira o navegador automaticamente.

Para criar atalho na Area de Trabalho:
1. Clique duas vezes em instalar-atalho-area-trabalho.bat
2. Use o atalho "Locacoes" criado na Area de Trabalho.

O app tambem abre pelo index.html, mas o modo servidor local ativa melhor os recursos PWA/offline.

Principais modulos:
- Painel executivo
- Imoveis
- Clientes
- Corretores
- Agenda/calendario
- Pipeline
- Contratos
- Financeiro
- Manutencao
- Vistorias
- Tarefas
- Documentos
- Relatorios
- Gestao de backup/importacao
- Sincronizacao Supabase com login e RLS

Supabase:
1. Crie um projeto no Supabase.
2. Rode o arquivo supabase-schema.sql no SQL Editor.
3. Em Authentication, crie um usuario com e-mail e senha.
4. No app, abra Gestao e informe Project URL, anon public key, e-mail e senha.

Os dados ficam salvos no navegador deste dispositivo. Use Gestao > Baixar backup JSON antes de grandes alteracoes.
