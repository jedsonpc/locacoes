LOCACOES - GESTAO DE EXCELENCIA

Forma recomendada de acesso:
1. Execute instalar-atalho-windows.bat.
2. Use o atalho "Cupe Beach Living" criado na Area de Trabalho ou no Menu Iniciar.
3. Com internet, o atalho abre diretamente a versao oficial na Vercel.
4. Sem internet, a copia local instalada fica disponivel como contingencia.

O instalador copia o app para %LOCALAPPDATA%\LocacoesApp, cria os atalhos e abre o sistema em modo aplicativo.

Acesso online:
- App oficial: https://locacoes-publish.vercel.app/
- Supabase usuarios: https://supabase.com/dashboard/project/afwynugokyejfbrtyobk/auth/users

O acesso pela Vercel e o modo principal. A copia local deve ser usada apenas quando o acesso online estiver indisponivel.

Principais modulos:
- Painel executivo
- Imoveis
- Clientes
- Corretores
- Agenda/calendario
- Pipeline
- Contratos
- Financeiro
- Manutencao
- Vistorias
- Tarefas
- Documentos
- Relatorios
- Gestao de backup/importacao
- Sincronizacao Supabase com login e RLS
- Secao Institucional com Sobre, Suporte, Privacidade e LGPD

Supabase:
1. O projeto gestao-locacoes ja esta configurado para este app.
2. O login inicial usa usuarios cadastrados em Authentication > Users.
3. Apos entrar, a sincronizacao em nuvem fica ativa para o usuario validado.

Os dados ficam sincronizados pelo Supabase e tambem podem ser mantidos no navegador. Use Gestao > Baixar backup JSON antes de grandes alteracoes.
