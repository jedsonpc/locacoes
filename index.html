Acesso ao sistema

O login e as senhas agora sao gerenciados exclusivamente pelo Supabase. As senhas antigas salvas neste navegador sao removidas automaticamente ao abrir o app.

Sincronizacao online

Os dados sao sincronizados automaticamente no Supabase quando houver login online. O endpoint HTTPS abaixo fica como alternativa manual.

Backups locais

O app salva copias automaticas neste navegador. O botao Baixar backup funciona em qualquer navegador; a gravacao direta em D:\App\backups depende de permissao do navegador.

Nenhum backup local gerado. Endereco local: localStorage:gestao-locacoes-backups-v1

Atualizacao do app

Quando houver uma nova publicacao no GitHub Pages, o app pode buscar e aplicar a versao mais recente automaticamente. Para outro computador, baixe o pacote desta versao e substitua os arquivos do app na maquina de destino.
